Configuración
Pasos para conectar este panel con tu Google Sheet.
1
Abre Apps Script en tu Google Sheet
En el Sheet: Extensiones → Apps Script. Borra el código por defecto y pega el archivo ranking-script.gs que tienes en tu proyecto.
2
Despliega como Web App
En Apps Script: Implementar → Nueva implementación
· Tipo: Aplicación web
· Ejecutar como: Yo
· Quién tiene acceso: Cualquier usuario
· Clic en Implementar → copia la URL que aparece
3
Pega la URL aquí abajo
En el panel izquierdo de esta página hay un campo "URL del Web App". Pégala ahí y guarda. A partir de ese momento este panel se comunica con tu Sheet.
4
Crea las pestañas del Sheet (primera vez)
Desde el Sheet abre el menú ⚽ Mundial → Crear pestaña RESULTADOS. Esto precarga los 72 partidos de grupos. Luego Crear pestaña RANKING.
⚠️ La contraseña del panel se define en la primera línea del script .gs (ADMIN_PASSWORD). Cámbiala por una que solo conozcas tú y vuelve a desplegar el Web App.
5
Guarda este archivo en GitHub (carpeta raíz)
Sube admin.html a tu repo. Los usuarios no saben que existe — no hay ningún link hacia él desde el sitio. Para acceder tú: tu-sitio.github.io/admin.html
